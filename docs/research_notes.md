# 股票筛选策略研究笔记

## 核心技术指标

### 1. 移动平均线 (Moving Averages)
- **SMA (简单移动平均)**: 计算特定周期收盘价的平均值
- **EMA (指数移动平均)**: 对近期价格赋予更大权重，反应更快
- **应用**:
  - 趋势识别: 价格在上升的MA之上 = 上涨趋势
  - 金叉信号: 短期MA上穿长期MA (如50日EMA上穿200日EMA)
  - 死叉信号: 短期MA下穿长期MA

### 2. MACD (移动平均收敛散度)
- **组成**:
  - MACD线: 12日EMA - 26日EMA
  - 信号线: MACD线的9日EMA
  - 柱状图: MACD线与信号线的差值
- **应用**:
  - MACD线上穿信号线 = 买入信号
  - MACD线下穿信号线 = 卖出信号
  - 零轴上方 = 多头力量增强
  - 背离: 价格创新高但MACD未确认 = 可能反转

### 3. RSI (相对强弱指数)
- **计算**: 14周期，范围0-100
- **应用**:
  - 强上涨趋势: RSI在50-80之间波动
  - 强下跌趋势: RSI在30-50之间波动
  - RSI > 70: 超买
  - RSI < 30: 超卖
  - 背离信号: 价格创新高但RSI未跟随 = 动能减弱

### 4. OBV (能量潮)
- **原理**: 成交量先于价格变动
- **计算**: 
  - 收盘价上涨: OBV = 前日OBV + 今日成交量
  - 收盘价下跌: OBV = 前日OBV - 今日成交量
- **应用**:
  - OBV与价格同向 = 趋势确认
  - OBV背离 = 潜在反转信号
  - 盘整期OBV上升 = 可能突破

## 早期上涨信号筛选条件

### 趋势启动信号
1. 价格突破关键移动平均线 (20日、50日)
2. 金叉形成 (短期MA上穿长期MA)
3. MACD金叉 (MACD线上穿信号线)
4. RSI从30附近反弹，突破50
5. 成交量放大配合价格上涨
6. OBV趋势线突破

### 动量确认信号
1. ADX > 25 表示趋势强度足够
2. 价格创52周新高或接近新高
3. 相对强度 (RS) 相对大盘表现优异
